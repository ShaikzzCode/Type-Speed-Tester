# ✨ About The Project

**Type Speed Tester** is a lightweight web application that allows users to test their **typing speed and accuracy** in a simple and distraction-free interface.

This project was built as a **basic beginner-level front-end project** to demonstrate how **HTML, CSS, and JavaScript** can work together to create an interactive typing test.

It focuses on **clean design, simplicity, and ease of use**, making it ideal for beginners who want to understand how small web applications work.

# 🖥️ How It Works

1. A text prompt appears on the screen
2. Start typing in the input field
3. The app tracks your typing progress
4. Your typing speed can be calculated based on performance

This helps users **practice typing and improve speed over time**.
